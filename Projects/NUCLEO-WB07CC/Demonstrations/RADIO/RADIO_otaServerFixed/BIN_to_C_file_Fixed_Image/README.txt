/* ****** How to create .c file for RADIO OTA Server FixedImage configuration ****** */
/* ********************************************************************* */

The script convert_bin2c_fixed_ota.exe generates a .c file to be used by the project demonstration RADIO/RADIO_otaServerFixed. The script must be used as follows:

convert_bin2c_fixed_ota.exe -i file_name.bin -o file_name.c

So, for example, the following line shows as generate the .c file from the binary generated by the project RADIO/RADIO_otaTx:

convert_bin2c_fixed_ota.exe -i RADIO_otaTx.bin -o RADIO_otaTx.c

For much details to how create a binary file to be used for the RADIO OTA operations, see the User Manual UM2736 - Radio Driver.
The example RADIO/RADIO_otaTx can be used as reference for generating such binary files.
